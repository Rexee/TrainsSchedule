### Trains Schedule
###### Anroid application. Show schedule of suburban electric trains using [Yandex API][yandex]

![Main][Main]

##### Used libraries
[Robospice][Robospice] - library for building network async apps  
[Retrofit][Retrofit] - library for working with REST API
